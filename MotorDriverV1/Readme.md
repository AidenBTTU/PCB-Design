The motor driver pcb designed in this folder was designed to operate with 6V TT Motors from a Raspberry Pi Pico. A signal voltage taken off a shunt resistor is amplified on board to be sent to a seperate overcurrent detection board. The PCB is designed to be operated with 2-4 TT motors controlled in pairs. Connectors are designed with the Raspberry Pi Pico in mind. Current through motors is not designed to handle more than 2A collectively. LFWD & LREV, RFWD & RREV are exclusive pairs and should not be held high at the same time or else through-current is likely. Pull up resistors allow the board to be operated from a digital voltage supply, but signal pins have only been tested at 3.3V. Mosfets used are the readily available IRLZ44N and IRF4905. The 1 mohm resistor should be a power resistor, and increased to 500mohm or greater if using long or thin cables.
